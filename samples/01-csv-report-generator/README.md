# 01. CSV売上レポート自動生成ツール

ココナラサービス: [Claude Codeで業務ツール・Webアプリを爆速開発します](https://coconala.com/services/4138522)（30,000円相当）

## このサンプルが示すもの

「Claude Code開発代行」で実際に**5日 / 30,000円で納品できるレベル感**の成果物例です。

**ありがちな業務課題**:
> 毎月、店舗ごとの売上CSVをExcelで集計して、上長向けにレポートを作っている。
> 月初の半日が完全につぶれる。グラフも貼り付けが面倒。

**このツールがやること**:
- CSVを1ファイル渡すだけで、月次サマリ・カテゴリ別・商品TOP10・推移グラフ込みの**HTMLレポート**を自動生成
- ブラウザで開ける単一HTMLファイル（社内Slackやメールにそのまま貼れる）
- Chart.jsでグラフは動的レンダリング（依存ライブラリ最小）

## 使い方

```bash
python report.py --input sample_data/sales_2026q1.csv --output report.html
open report.html
```

引数なしでもサンプル実行可能:

```bash
python report.py
# → output/sample_report.html を生成
```

## 入力CSVのフォーマット

| date | product | category | quantity | unit_price |
|---|---|---|---|---|
| 2026-01-05 | カフェラテ | ドリンク | 12 | 480 |
| 2026-01-05 | チーズケーキ | フード | 5 | 620 |

`sample_data/sales_2026q1.csv` に3ヶ月分のダミーデータあり。

## 出力イメージ

`output/sample_report.html` を開くと以下のレイアウト:

1. **KPIカード** — 総売上 / 取引件数 / 平均単価 / 最終取引日
2. **月次売上推移** — 折れ線グラフ
3. **カテゴリ別売上** — 円グラフ
4. **商品別売上 TOP10** — 横棒グラフ + テーブル
5. **明細テーブル** — フィルタ・ソート対応

→ プレビュー: [output/sample_report.html](output/sample_report.html)

ココナラサービス: [Claude Codeで業務ツール・Webアプリを爆速開発します](https://coconala.com/services/4138522)（30,000円〜）の **LP制作版**

## このサンプルが示すもの

地域の小規模事業者（サロン・整体・カフェ・士業）向けの**完成度の高いLPテンプレート**を3パターン同梱。
1ファイル（HTML+インラインCSS+インラインJS）で完結するため、レンタルサーバや無料ホスティングにそのまま置けます。

## 同梱テンプレート

```bash

open templates/01-salon.html

xdg-open templates/01-salon.html

python3 -m http.server 8000
```

## 全テンプレートの共通機能

- **完全レスポンシブ** — スマホ/タブレット/PCで自然なレイアウト
- **CSS変数で配色カスタマイズ** — `--accent` 等を変えるだけで色を変更可
- **スクロールアニメーション** — IntersectionObserverで軽量実装
- **問い合わせフォーム** — Formspree / Google Forms / メール直リンクに対応可
- **構造化データ (Schema.org)** — SEOで店舗情報をリッチリザルトに表示
- **OGP対応** — SNSシェア時のサムネイル制御
- **SEO基本対応** — meta description、適切な見出し階層、画像alt
- **アクセシビリティ** — キーボード操作、スクリーンリーダー対応
- **依存ゼロ** — フレームワーク・ライブラリ不要、HTMLファイル1つで完結

## カスタマイズの仕方

```html
<!-- 配色を変える -->
<style>
  :root {
    --accent: #e5a662;     /* メインカラー */
    --accent-dark: #c4884a; /* ホバーカラー */
    --bg: #fdfaf6;          /* 背景 */
    --ink: #2d2419;         /* テキスト */
  }
</style>
```
